事件冒泡

取消事件冒泡

event.stopPropagation();
event.cancelBubble=true;

事件冒泡的优势

减少代码量

得到event对象(事件对象)

1.html代码中直接使用event

<input type="button" value="button"
onclick="alert(event.clientX+':'+event.clientY);event.stopPropagation();">

开发中,一定是html代码和js代码分离

所以上述写法意义不大

2.在js代码中使用event

function fn1(){
    //ie和chrome中可以直接使用event对象
    alert(event.clientX+'可以直接使用event对象');
}

需要考虑兼容问题

IE浏览器支持,后期新版本chrome对这种使用event的方式做了兼容

在js代码中,直接使用event对象的方式,适用于IE和chrome

FireFox如何在js中获取使用event对象

火狐使用上述方式直接使用event对象会报错

evetn is not defined的错误

function fn2(event){
    alert(event.clientX+'ie,chrome,ff都行');
}

总结

在函数中调用event对象,使用传参的形式

ie chrome ff 都认可

开发中要使用这种模式

使用event对象

1.event对象中保存着很多属性和方法

2.event对象获取事件源

2.1 适用于IE和chrome获取事件源方式
    event.srcElement;
    旧版本chrome不兼容这种方式.后期做了更新

        function fn3(event){
            var obj=event.srcElement;
            console.log(obj);
        }

2.2 firefox---event.target;

        function fn4(event){
            var obj=event.target;
            console.log(obj);
        }
        //event.target;
        //适用于ff,chrome,新版本ie
        //不适用于IE8及其以下版本

总结,浏览器兼容

var obj=event.srcElement || event.target;

    <script type="text/javascript">
        function cal(event){
            var oP=document.getElementById("screen");
            //获取事件源
            var obj=event.target|| event.srcElement;
            //console.log(obj.nodeName);
            if(obj.nodeName!="INPUT"){
                return;
            }
            var value=obj.value;
            //console.log(value);
            if(value=="C"){
                //清空P
                oP.innerHTML="";
            }else if(value=="="){
                //计算p中string,再在p中显示结果
                try{
                    var exp=oP.innerHTML;
                    var result=eval("("+exp+")");
                    oP.innerHTML=result;
                }catch(e){
                    oP.innerHTML="Error";
                }
            }else{
                //在p中追加value
                oP.innerHTML+=value;
            }
        }
    </script>

jQuery是一个JavaScript库

极大的简化了JavaScript编程

<body>

     value="变大" onclick="bigger();">
    <p>jQuery是一个JavaScript 库</p>
    <p>jQuery极大的简化了JavaScript编程</p>
    <p>jQuery 非常容易学</p>
</body>

<script type="text/javascript" src="../jquery-3.1.1.min.js"></script>
<script type="text/javascript">
    function bigger(){
        //获取当前p中字号
        var size=$("p").css("font-size");
        size=size.replace("px","");
        //字号变大,设置给p的样式
        $("p").css("font-size",++size+"px");
    }
</script>

jQuery对象详解

总结:带有jQuery方法的Dom数组,称之为jQuery对象

只有jQuery对象可以调用jQuery方法

DOM对象可以调用DOM方法

jQuery—>dom obj[index]

DOM—>jQuery $(dom)
